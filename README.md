# AI-Based Personal Finance Budgeting & Expense Tracker

A full-stack intelligent finance management system designed to help users **track expenses, analyze historical spending, and manage personal budgets efficiently**. The platform leverages rule-based and AI-assisted categorization to automatically organize expenses into meaningful groups such as **Food, Travel, and Entertainment**, enabling data-driven financial decisions.

---

## Overview

Managing day-to-day expenses manually is often inefficient and error-prone. This project provides a centralized web application where users can securely record transactions, visualize spending trends, and gain actionable insights into their financial behavior.

The system focuses on:

* Real-time expense tracking
* Automated categorization
* Historical analysis
* Clean and responsive user experience

---

## System Architecture

The application follows a standard **full-stack web architecture**:

* **Frontend:** Handles user interaction and visualization
* **Backend (Django):** Processes business logic and authentication
* **Database:** Stores user and transaction data
* **Analytics Layer:** Performs categorization and summary computations
